This is an advanced risk management tool designed for institutional investment portfolios. Instead of assuming the stock market is completely predictable, this program runs a Monte Carlo simulation meaning it calculates 1,000 separate, randomized future paths for a $10 million portfolio over a 10 day trading period.

The core focus of this tool is stress testing. The code introduces a sudden, unexpected "Carbon Tax Shock" (a regulatory event that slashes carbon heavy assets by 8% overnight). It then calculates the 95% Value at Risk (VaR), which tells a fund manager the absolute maximum amount of money the portfolio could lose under normal market conditions versus a sudden green regulatory crackdown. It proves how ignoring environmental policies creates hidden, massive financial risks for investment funds.
